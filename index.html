HTML+CSS+Javascript

  • HTML5

    • HTML5(Hoper text Markup Lenguaje, version 5) es la quinta revision del lenguaje basico de la World Wide Web World Wide Web HTML

    Usa la extension.html

  • CSS3

    • Hoja de estilo en casacada o CSS(siglas en ingles de cascading stylr sheets) es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML.

    Usa la extencion css

  • JavaScript

    • JavaScript (Abreviado comunmente"Js") es un lenguaje de programacion interpretado utilizando principalmente en el lado del cliente(client.side), implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y painas web dinamicas.

    Usa la extencion.js
